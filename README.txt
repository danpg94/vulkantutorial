Project created in Linux (Ubuntu)

Requirements:
    - Install using apt or dnf:
        g++ make vulkan-tools vulkan-tools vulkan-validationlayers-dev spirv-tools libglfw3-dev libglm-dev libxxf86vm-dev libxi-dev
    - Download the shader compiler from https://github.com/google/shaderc
    - Extract file from compressed folder and copy install/bin/glslc to /usr/local/bin
